이미 여러 편의 논문(총 15편)을 함께 보내주셨습니다. 요청하신 대로 각 논문에 대해 (1) 전문적 요약과 (2) 중학생용 쉬운 설명을 모두 포함한 구조화된 분석을 제공할 수 있지만, 진행 전에 중요한 점과 선택지를 확인드려야 합니다.

중요 사항(현재 환경 한계)
- 저는 직접 웹을 탐색하거나 외부 PDF를 다운로드할 수 없습니다. 따라서 귀하가 제공하신 arXiv 링크를 제가 열람해 전체 전문을 읽을 수는 없습니다.
- 전체 논문 원문(Full PDF)을 직접 분석하여 "논문 목차 기반으로 구조화된 요약"을 제공하려면, 논문 PDF 파일들을 여기 채팅에 업로드해 주시거나, 원문 텍스트를 붙여 주셔야 합니다.
- 만약 업로드가 어렵다면, 제가 제공할 수 있는 대안은 각 논문의 abstract·제목·메타데이터를 바탕으로 한 요약·분석입니다.이는 전체 원문을 읽고 목차별로 상세히 정리한 결과보다는 범위가 좁습니다.

진행 옵션 — 하나를 선택해 주세요
1) 전체 원문 기반 정밀 분석(권장)
   - 사용자가 각 논문의 PDF 파일을 이 채팅으로 업로드.
   - 업로드된 원문을 목차(Contents) 기준으로 따라가며,
     - 요약(Professional): 목차별 주요 내용, 방법론, 실험, 결과 및 한계, 시사점, 활용방안 등 상세 정리
     - 쉬운설명(Simple): 중학생 수준의 비유와 핵심 아이디어 설명
     - 관련분야, 추천수(사용자가 제공한 upvotes 사용 가능), PDF 다운로드 링크(원문 업로드 시 내부 링크 표기)
   - 모든 논문에 대해 위 형식으로 제공.

2) 초록(abstract) 기반 요약 (빠름)
   - 제공된 abstract·제목·링크·썸네일만을 바탕으로 요약 제공.
   - 목차 기반의 깊은 세부 정리는 불가하지만, 주요 기여, 방법의 핵심, 주요 결과·시사점을 전문·쉬운 설명 두 가지로 정리.
   - 빠르게 전체 15편에 대해 응답 가능.

추가 질문(선택사항)
- 한 번에 모든 논문(15편)을 정리할까요, 아니면 우선 몇 편(예: 상위 3편)만 집중해서 진행할까요?
- PDF 업로드 가능하신가요? (가능하면 업로드 방법을 안내드리겠습니다.)
- abstract 기반 요약으로 진행하면, 원하시는 분량(예: 각 논문당 전문 요약 400–800자, 쉬운설명 150–250자 등)을 알려주세요.

제안 사항
- 만약 원문 업로드가 어렵다면, 우선 abstract 기반 요약(옵션 2)으로 전체 15편을 빠르게 제공해 드리고, 사용자가 특정 논문에 대해 원문을 업로드하면 그 논문만 심층 목차 기반 분석으로 재작성해 드리는 절충안을 권합니다.

원하시는 진행 방식을 알려주시면, 그에 맞춰 바로 작업을 시작하겠습니다.
