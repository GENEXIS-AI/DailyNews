요청하신 13편의 논문에 대해 "제목 / 요약(전문가용) / 쉬운설명(중학생용) / 관련분야 / 추천수 / PDF 다운로드 링크 / 썸네일" 형식으로 상세하게 정리해 드리려면 논문 본문(특히 목차·섹션별 내용)에 직접 접근하여 전체를 읽고 TOC(목차) 구조에 따라 요약하는 것이 필요합니다.

현재 환경에서는 외부 PDF 링크를 자동으로 열람하여 본문 전체를 읽을 수 없습니다. 따라서 다음 중 하나를 선택해 주세요:

옵션 A — 초록(제목·초록·메타데이터) 기반 요약(지금 바로 가능)
- 제공하신 arXiv 메타정보와 초록을 바탕으로 각 논문에 대해 전문가용 요약(요지), 중학생용 쉬운설명, 관련분야, 추천수, 제공하신 PDF 링크·썸네일을 포함한 형태로 정리해 드립니다.
- 다만 논문 내부의 상세한 절(예: 방법론 세부 수식, 실험 설정의 모든 하이퍼파라미터, 목차별 내용 등)을 반영한 TOC 기반 심층 요약은 제한됩니다.
- 처리 완료 시간: 즉시(한 번에 모두 정리).

옵션 B — 논문 원문을 업로드해 주시면 TOC(목차) 구조에 따라 심층 정리 제공
- 사용자가 논문 PDF 파일(또는 arXiv 원문 복사본)을 직접 업로드해 주시면, 각 논문의 목차를 따라 상세 요약(섹션별 핵심 아이디어, 기법, 실험, 한계, 활용·시사점 등)과 중학생용 쉬운설명을 함께 제공합니다.
- 처리 완료 시간: 각 논문당 분량에 따라 다르지만, 전체 13편이면 단계적으로(예: 1–3편씩) 진행 제안 가능.

옵션 C — 우선 1편(예: OpenWorldLib)만 TOC 기반으로 심층 요약을 원하시면 해당 논문 PDF 업로드 또는 복사본 제공
- 한 편씩 자세히 처리하는 방법입니다.

추가 확인사항 (선택사항)
- 모든 논문을 한꺼번에 초록 기반 요약으로 받으실지(옵션 A), 아니면 먼저 몇 편만 심층 분석할지(옵션 B/C) 선택해 주세요.
- 요약 분량(간단 요약 vs. 섹션별 상세 요약) 선호가 있으시면 알려주세요.
- 출력 형식은 요청하신 템플릿(썸네일, 제목, 요약, 쉬운설명, 관련분야, 추천수, PDF 링크)으로 맞춰 제공하겠습니다.

원하시는 옵션을 알려주시면 즉시 시작하겠습니다.
