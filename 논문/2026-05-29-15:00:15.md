안녕하세요. 요청하신 논문 요약을 바로 진행하려면 몇 가지 정보가 필요합니다. 아래 항목을 제공해 주세요.

필수 정보
- 논문 파일(PDF) 업로드 또는 DOI / arXiv ID / 공식 URL / 논문 제목+저자

선택 정보 (요약 커스터마이즈)
- 전문(academic) 요약과 쉬운설명(중학생 대상)을 둘 다 원하시는지, 아니면 먼저 어느 쪽을 받을지
- 요약 분량(예: 전문 요약 — 상세(섹션별, 수식·그림 설명 포함) / 간단(요점 중심))
- 중점적으로 알고 싶은 부분(예: 방법론, 수식 유도, 실험 설정·결과, 응용/임플리케이션, 관련 연구 등)
- 표나 수식 설명을 자세히 원하는지 여부
- 영어 원문을 함께 참고할지 여부(원문 인용·원문 문장 표기 요청 시)

저작권 관련 안내
- 전체 원문을 그대로 복제해서 제공할 수는 없습니다. 다만 논문 링크(PDF)가 공개되어 있다면 다운로드 링크를 제공하거나, 제공해주신 PDF를 바탕으로 상세 요약·해석을 드릴 수 있습니다.

응답 형식 (예시)
- 제공하신 논문을 바탕으로 아래 형식으로 정리해 드립니다.
  - 제목, 썸네일(가능한 경우), 전문 요약(목차별), 쉬운설명(중학생 대상), 관련분야, 추천수, PDF 다운로드 링크(가능한 경우)

진행 방법 예시 질문(원하시면 바로 복사해서 답장하세요)
1. 논문 PDF 업로드 (또는 DOI/URL): 
2. 전문·쉬운설명 둘 다 원함? (둘 다 / 전문만 / 쉬운설명만)
3. 특별히 중점 설명 원하는 섹션:
4. 원하는 분량(예: 전문 800~1500단어, 쉬운설명 200~400단어):

원하시는 논문 파일이나 식별자를 보내주시면, 요청하신 형식(전문 요약 + 쉬운설명)을 기준으로 목차에 따라 상세하고 이해하기 쉬운 한국어 요약을 준비하겠습니다.
