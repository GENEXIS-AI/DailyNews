## 제목: OPUS: Towards Efficient and Principled Data Selection in Large Language Model Pre-training in Every Iteration
**요약**:
OPUS는 대규모 언어 모델의 사전 훈련에서 효율성을 높이기 위해 데이터 선택 프레임워크를 제공합니다. 이 프레임워크는 최적화기 유도 업데이트 투영에 기반해 데이터 후보를 점수화하여 계산 비용을 줄이면서도 우수한 성능을 달성합니다. 각 반복(iteration)마다 안정적인 대리 파생 목표 공간(stable proxy-derived target space)에 데이터를 매핑하여 사전 훈련의 효율성을 극대화합니다.

**쉬운설명**:
이 연구는 대규모 언어 모델이 데이터를 더 똑똑하게 선택할 수 있도록 도와주는 OPUS라는 도구를 소개합니다. 쉽게 말하면, 이 도구는 어떤 데이터를 사용할지 더 현명하게 결정하여 계산할 일을 줄이고도 높은 성능을 내게 합니다.

**관련분야**:
기계 학습, 자연어 처리, 데이터 분석
