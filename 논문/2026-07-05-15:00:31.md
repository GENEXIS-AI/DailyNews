요청해 주신 논문 목록(총 10편)에 대해 “제공받는 전체 논문 전부 분석”하여 형식(썸네일, 제목, 요약(전문), 쉬운설명(중학생용), 관련분야, 추천수, PDF 링크)으로 작성하려면 원문(또는 최소한 전문 PDF/목차/초록)이 필요합니다.

중요 안내(정확성·저작권 관련)
- 현재 제 세션은 외부 웹사이트를 직접 방문하거나 파일을 자동으로 내려받을 수 없습니다. 따라서 제가 논문 전체를 정확히 분석하려면 사용자가 PDF 파일들을 업로드하시거나 논문 전문/목차/초록을 붙여 넣어 주셔야 합니다.
- 논문 전체 텍스트를 제공해 주시면 저작권을 침해하지 않는 범위 내에서 요청하신 구조로 전문 요약과 쉬운 설명을 모두 제공합니다.
- 만약 파일 업로드가 불가능하시다면, 각 논문의 초록(abstract)과 목차만 제공해 주셔도 어느 정도 구조화된 요약을 만들 수 있습니다. 다만 깊이 있는 기술적 세부사항(수식, 실험 세팅, 결과 해석 등)은 전문이 필요합니다.

선택 가능한 작업 방식 (원하시는 옵션 번호 알려 주세요)
1) PDF 직접 업로드: 각 논문 PDF 업로드 → 제가 전체 분석 후 지정된 형식으로 출력 (권장)
2) 초록/목차 제공: 각 논문의 abstract와 table of contents(혹은 TOC를 복사해 붙여넣기) 제공 → 해당 정보 기반으로 구조화 요약 작성(일부 기술적 상세 제한)
3) 메타데이터 기반 간단 안내: 제목·링크·추천수·썸네일만으로 가능한 범위의 표층 요약 및 예상 관련분야(정확성 낮음). 논문 전체 분석 요구 조건을 충족하지 못함.
4) 한 편씩 진행: 우선 1편을 업로드 → 시범 요약 제공 후 나머지 진행 여부 결정.

추가 질문
- 모든 논문에 대해 "요약(전문)"과 "쉬운설명(중학생용)"을 한 번에 제공할까요, 아니면 먼저 전문 요약만 받을지 선택하시겠습니까?
- 출력 길이/상세도(예: 각 논문당 전문 요약 800–1200단어, 쉬운설명 150–300단어 등) 선호가 있으시면 알려 주세요.
- PDF 업로드 방식을 모르시면 업로드 방법(파일 붙여넣기/드래그앤드롭 등) 안내해 드립니다.
