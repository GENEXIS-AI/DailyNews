중요 안내
- 현재 저는 외부 URL로부터 직접 PDF 파일을 자동으로 내려받아 분석할 수 없습니다. 따라서 논문 전문(full PDF)을 기반으로 한 심층 분석을 원하시면 각 PDF 파일을 업로드해 주시거나, 본문 텍스트를 붙여 넣어 주셔야 합니다.
- 만약 업로드가 불가능하다면, 지금 제공해주신 메타데이터(제목·초록·썸네일·pdf_link)를 바탕으로 요약·해석을 제공할 수 있습니다. 이 경우 논문의 세부 섹션(예: 방법론 수식, 실험 표, 세부 아키텍처 등)에 대한 깊이있는 검토는 제한됩니다.

진행 옵션 (선택해 주세요)
1) 초록(abstract)·메타데이터 기반 요약 — 모든 논문(목록 전체)에 대해 즉시 "전문(학술적) 요약" + "쉬운설명(중학생용)" 형태로 각 논문별로 제공. (빠름, 다수 논문 가능, 세부 기술적 검증·수식 분석 불가)
2) PDF 기반 심층 분석 — 사용자가 각 논문 PDF를 업로드 → 논문 목차(TOC) 구조에 따라 전문 요약 + 쉬운설명 + 세부 섹션별 분석(방법, 이론, 실험, 한계, 활용법 등) 제공. (정밀함 높음, 업로드 필요)
3) 우선순위 처리 — 목록에서 우선순위(예: 상위 5개만 심층 분석 또는 상위 10개 초록요약, 나머지는 간단요약)를 지정해서 단계적으로 처리.
4) 출력 형식 관련 — 요청하신 Response Format(썸네일·제목·요약·쉬운설명·관련분야·추천수·PDF 다운로드 링크)을 그대로 따릅니다. 각 논문마다 "요약"과 "쉬운설명" 두 종류를 함께 제공합니다. 한 번에 모든 논문을 한 문서로 묶을지, 논문별로 개별 항목으로 출력할지 선택해 주세요.
